How to use the helper script for cataloging the DZL collection in LibraryThing!

  1. Add this link as a bookmark:
    • DZL cataloging helper
  2. Log into LibraryThing.
  3. Go to the manual entry page under the "Add books" tab.
  4. Load the bookmark!

What does this do?

It makes it easier to add zines into the DZL catalog by reformatting LibraryThing's manual entry page. You can now:

  • Add covers to zines directly
  • Search and copy data from existing zines
  • Click to select common zine dimensions
  • Automatically generate a physical summary
  • Catalog multiple items without leaving the page
Other improvements are focused on making it easy to follow the DZL cataloging rules:
  • Hides unused fields
  • Syncs "Collections" with tags and the "Other call number" field
  • Syncs comment and summary fields
  • Suggests tags using the approved vocabulary list
  • Automatically copies volume and issue information from titles to the comment and summary fields

Problems?

Please report any issues you have! 🙂 You can email, use the DZL Slack channel, file a bug report, whatever you'd like! This was originally tested using Chrome on a Mac laptop and may have several bugs on other platforms.
